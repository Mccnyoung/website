Problem

01 Label text in text fields disappears when tapped

The user ID can be entered using either a phone number or email address so it's critical to deliver. However, it disappears when the user taps the text field.

02 Excessive links in the login support menu

The Gopax login support menu includes not only essential links such as "Find ID" and "Reset Password," but also an overwhelming number of overly detailed options (e.g., Sign up for personal, corporate) and instructional texts.

03 Lack of feedback on PIN page authentication success

After entering the PIN, feedback on the success of the authentication is not properly provided.

Solution

01 Labels don't disappear when tapped

The label text now moves upward in a smaller size when the user taps the field, ensuring that the information remains visible and easily recognizable.

02 Simplified login support menu

After analyzing GA data, we found that the click rate for corporate sign-up was only 1%, so we merged personal and corporate sign-ups into a single "Sign Up" option.

03 Add animation to indicate authentication success

To provide intuitive feedback on authentication success, an animation was created and delivered to the developers as a JSON file for successful implementation.
